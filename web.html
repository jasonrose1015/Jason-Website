My Web Projects - From Beginner To Slightly Skilled :)

I didn't get started in web design until I was 34 years old. By that time, most people have already published projects and have a great portfolio. I have been working a little each day, but trying to learn everything involved in web design can be a little overwhelming. It is difficult to find good resources that explain the "why" in a lot of projects. I wish I would've taken these classes in college where I could have collaborated with others more often. Now, I'm stuck reading and following tutorials in books and watching YouTube videos. It is still a lot of fun. I love learning new skills in web design. Being able to use code and create something amazing is an awesome skill to know. But I also love anything related to tech! If you are new to tech or web design, I encourage you to dive in and give it a shot. It is a challenge, but so much fun!

Why I love to code

I started to get into HTML and CSS about four years ago when I started teaching web design for a local high school. Having very little experience in HTML and CSS, I was doing a lot of learning myself each day.

I did a lot of research before classes started in an attempt to find the best projects and lessons for the students to learn from. After a couple of days of research, I came across IEClass. I used this website to teach myself so that I would be prepared for any student questions. Once I completed all of the projects, I moved on to Jon Duckett's HTML and CSS book. Along with his JavaScript book, I am currently working on CSS Grid and CSS Flexbox.
