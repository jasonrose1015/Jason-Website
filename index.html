My Latest in Coding - January 2019

All of the books that I have listed to the right are great for both entry level and advanced devlopers. The Gamer's Guide to Coding is the perfect book to start with. It explains programming and uses interesting examples specifially created for gamers to get them interested. The Big Nerd Ranch Guide to Swift Programming is an excellent way to start producing iOS apps. Like all books on this list, it is "new coder" friendly and easy to follow. The Adobe Photoshop Manual is obviously not a coding book. However, it shows you how to master photo editing, which is just as important in web design and programming.
